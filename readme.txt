== Description ==

Radin App is plugin for Create application android for Woocommerce in time 5 minutes
Radin App Created application android is native

Radin App is an application builder plugin for the WordPress platform that is compatible with the WooCommerce, Store and Digits plugin, which allows you to have less than 5 minutes of a store application similar to the Digikala application for Android natively (not as a web view)

After downloading the Radin App plugin on your website, you can download your app instantly with just a few simple clicks and selecting the name and color of the app.

Some features of Radin App plugin:
1.compatible with dokkan plugin for Multiple sales
2.compatible with digits plugin for login with phone number
3.Compatible with all payment gateways
4.Advanced Product Filter
5.Send notifications
6.Build a dynamic home screen

The process of starting work with Radin App plugin:
1.Download and install the plugin on the WordPress site
2.in the radin app plugin - Go to the Home and  Designing Home section
3.enter the appname color logo and etc then click on the download button
4.please wait 5 - 10 minute to compile android application for you
5.finish... please copy app_id and auth to use it for notification section
6.please download the application and use it
7.injoy
